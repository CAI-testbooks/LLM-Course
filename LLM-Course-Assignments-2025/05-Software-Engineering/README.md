# 方向05：智能软件工程综合作业

## 题目描述
构建一个AI驱动的软件开发助手，支持从需求到代码生成、测试或调试等智能开发助手（类似DevAgent或SWE-Agent）。

## 题目要求
- **数据准备**：使用HumanEval、MBPP、SWE-Bench等代码基准。
- **模型选择**：基于代码专用LLM（如DeepSeek-Coder、Qwen3-Coder）构建Agent。
- **核心功能**：需求理解、代码生成、单元测试自动编写、Bug修复等。
- **迭代优化**：加入反思机制、工具调用。
- **部署**：构建CLI或Web工具。
- **评估**：在SWE-Bench或自定义任务上的成功率。
