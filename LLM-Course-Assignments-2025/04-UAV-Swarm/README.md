# 方向04：无人机集群综合作业

## 题目描述
实现一个多无人机（UAV Swarm）协同系统，支持集群巡检、搜索救援或编队飞行。

## 题目要求
- **数据准备**：使用模拟数据集或AirSim/CARLA生成轨迹数据。
- **模型选择**：基于多智能体强化学习（MARL）或LLM驱动规划（如Agent协调）。
- **核心功能**：分布式路径规划、避障、任务分配、通信模拟。
- **迭代优化**：更新奖励函数、通信策略，提升覆盖率与鲁棒性。
- **部署**：在AirSim/Gazebo模拟环境中运行多UAV Demo。
- **评估**：覆盖效率、碰撞率、任务完成时间。
