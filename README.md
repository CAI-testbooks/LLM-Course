# CWRU数据分析代理系统

基于LLM Agent和DSPy框架的自动化数据分析系统，针对CWRU轴承故障诊断数据集为基础的大模型故障诊断分析。

## 特性

- 🤖 **AI驱动分析**: 使用LLM Agent自动规划和分析流程
- 🔄 **全自动化**: 从数据清洗到报告生成的全流程自动化
- 📊 **多维分析**: 支持统计检验、可视化、机器学习建模
- 📈 **智能洞察**: 基于DSPy的智能洞察生成
- 🎨 **交互式界面**: 支持命令行交互模式
- 📋 **报告输出**: 自动生成Markdown和PDF报告
